# End to End ML Project

### Create a project folder
```bash
mkdir <"project folder name">
```

### Initialise git
```bash
git init
```

### Add file to stage
```bash
git add <"file name">
```

### Commit fils to git
```bash
git commit -m <"commit message">
```

### Push the branch to remote github
```bash
git push -u origin <"branch name">
```

### Run init_setup.sh
```bash
bash <'filename'>.sh
```
